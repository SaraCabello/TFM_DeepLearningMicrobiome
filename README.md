# Deep Learning Microbiome Project Repository
Repository to store data, models and scripts used and obtained during my master's final project

#### Data
This folder contains data used to train the models (original_data) and data obtained when this original data is transformed using the different models built along this work.

Transformed data folder is organised into sub-folders according to the models that produce each resulting file. 
  
#### Models
All the trained models obtained during the development of the project are available here, organised in different folders according to the different types of models of this work.

#### Scripts
R and Python scripts used to preprocess data, build the models and analyse the results are found in this folder, numbered according to the order they were applied along this project.

#### Figures
This folders contains all plots and figures present in the project document (not available here), named in an analogous way as in it.
